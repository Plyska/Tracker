# Tracker — Frontend

React 19 + TypeScript + Vite. Загальний опис проєкту й план — у кореневому [CLAUDE.md](../CLAUDE.md).

## Скрипти

```bash
npm install             # залежності
npm run dev             # dev-сервер → http://localhost:5173
npm run build           # продакшн-збірка (tsc + vite build)
npm run preview         # переглянути зібране
npm run lint            # ESLint
npm run storybook       # Storybook → http://localhost:6006
npm run build-storybook # статична збірка Storybook
```

Запуск через Docker — див. кореневий README та `docker-compose.yml`.
